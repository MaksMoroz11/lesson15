# lesson15
Класс FinanceManager — отвечает за управление финансами пользователя.
Методы:

_init_(self): инициализирует баланс и список транзакций.

add_transaction(self, type, amount, category): добавляет новую транзакцию 
(доход/расход), обновляет баланс и сохраняет данные в историю.

get_transactions(self): возвращает список всех транзакций.

get_balance(self): возвращает текущий баланс.

save_data(self): сохраняет данные в текстовый файл.

load_data(self): загружает данные из текстового файла.

run(self): отвечает за взаимодействие с пользователем. Выводит 
текстовое меню, обрабатывает команды (добавление доходов/расходов, 
просмотр баланса и транзакций, завершение программы). 
Этот метод управляет циклом работы программы.
